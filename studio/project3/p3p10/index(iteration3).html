Mondays

      ‘Mondays’ is a collection that demonstrates the highest temperature every Monday in Manhattan, New York, and Pasadena, Los Angeles from Weather Underground of 2021. The temperature would then be rounded to the nearest fifth and corresponded to a four by four-inch color box based on the key created by Gianna Waldron.

      The collection was designed by Gianna, who also collected the temperature data for New York. This project aims to show data and challenge designers to demonstrate data in a new way. I (Hallie Young), finished off the collection by adding weather data from Los Angeles, which is the city I grew up. This connects me and the collection, deepening the purpose of the project and showing any audience the temperature contrast between these two cities. I then transferred the data and designed this website into grid boxes giving this concept of calendars allowing others to view data collection.
